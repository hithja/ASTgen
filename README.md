# ASTgen
Abstract Syntax Tree generator on Python for C++
## Usage
First, create JSON config.
### Example:
```JSON
{
    "name": "Expr",
    "sys_includes": ["vector", "variant", "memory"],
    "user_includes": ["Token.hpp"],
    "types": {
        "Binary": "Expr left, Token oper, Expr right"
    }
}
```
### Config fields
- `name` - name of base class
- `sys_includes` - system header files import (`#include <LIB>`)
- `user_includes` - user header files import (`#include "LIB.hpp"`)
- `types` - AST types and their fields

See full example ![here](https://github.com/hithja/ASTgen/blob/main/Expr.json).
### Generating
Run:
```bash
python3 ./ASTgen.py CONFIG_NAME.json <output directory>
```
to generate the AST header file.
### Generated `Expr.hpp` (example)
```C++
/*
	- Generated by ASTgen
	- GenAst.py usage: python3 ./tools/ASTgen.py ./include/AST
*/
#pragma once
#include <vector>
#include <variant>
#include <memory>
#include "Token.hpp"

using Object = std::variant<std::string, std::monostate, double, bool>; 
                
class Expr {
    public:

};
    
class Binary : public Expr {
    public:
	std::shared_ptr<Expr> left;
	Token oper;
	std::shared_ptr<Expr> right;

	Binary(std::shared_ptr<Expr> left, Token oper, std::shared_ptr<Expr> right)
	: left(left),
	oper(oper),
	right(right) {}
};
```
